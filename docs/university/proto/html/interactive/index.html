Контекст проекта

На обзорном экране собраны быстрые ссылки, пояснения по структуре интерфейса и краткий контекст проекта.

Рабочая область проекта

Главный экран показывает основные рабочие зоны проекта в едином контексте: задачи, знания, заметки и общую навигацию по проекту. Интерфейс ориентирован на быстрый переход в рабочий раздел без перегрузки деталями.

Для дальнейшего взаимодействия пользователь может перейти к задачам через верхнее меню или открыть материалы базы знаний в соседнем разделе.

Контекстная инструкция

Используйте верхнее меню или карточки быстрого перехода. Главный экран нужен как точка входа: он показывает доступные рабочие контуры и помогает быстро перейти к задачам и документации.

Следующее действие

Откройте раздел задач, если нужно посмотреть очередь работ, либо перейдите в базу знаний, если требуется проектная документация.
